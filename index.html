Soil Moisture & Fertility Analysis

IoT-enabled system with Arduino and YOLOv8 analyzing 500+ soil images daily. Achieved 92% fertility detection and 88% moisture classification.

YOLOv8 Arduino Flask Python

The Soil Moisture and Fertility Analysis with Camera Integration project is a smart agriculture solution developed with the aim of improving farmers’ conditions through the use of modern technology.

Motivation

Many farmers still rely on traditional methods and personal experience to judge soil quality and moisture levels, which often leads to over-irrigation, under-irrigation, low crop yield, and unnecessary fertilizer usage. The motivation behind this project was to provide farmers with a simple, accurate, and affordable technological tool that can help them make better decisions regarding watering and soil treatment, ultimately increasing productivity while reducing resource wastage.

How It Works

The system combines IoT hardware, computer vision, and machine learning to analyze soil conditions in real time. An Arduino-based setup is used along with a camera module to capture soil images directly from the field. These images are then processed using YOLOv8-based deep learning models, which analyze visual patterns, texture, and color variations in the soil to determine its fertility level and moisture condition.

Key Metrics

On a daily basis, the system is capable of analyzing 500+ soil images, providing consistent and scalable monitoring. The machine learning model achieved 92% accuracy in soil fertility detection and 88% accuracy in moisture classification, ensuring reliable results for real-world agricultural use.

Farmer-Friendly Interface

To make the system farmer-friendly, the processed results are displayed through a web-based interface developed using Flask, HTML, and CSS. This interface presents clear insights such as:

  • Current soil moisture level
  • Soil fertility status
  • Recommendations for irrigation and fertilizer usage

Impact

By using accurate data instead of guesswork, farmers can avoid excessive watering and fertilizer application. As a result, the system helps in reducing water usage by 20–25%, lowering operational costs and promoting sustainable farming practices.

Overall, this project demonstrates how IoT and AI-driven solutions can empower farmers by providing them with actionable insights, improving crop yield, conserving natural resources, and supporting long-term agricultural sustainability.

Predictive Analytics in Autism

Deep Learning model designed with TensorFlow & Keras. Achieved 94% prediction accuracy for early diagnosis.

Deep Learning TensorFlow CNN

The Predictive Analytics in Autism Spectrum Disorder (ASD) project focuses on using artificial intelligence and deep learning techniques to support the early and accurate diagnosis of autism.

Motivation & Problem

Autism Spectrum Disorder is a neurodevelopmental condition that affects communication, behavior, and social interaction. Early diagnosis plays a critical role in enabling timely intervention. However, traditional diagnosis methods largely depend on behavioral observations and expert judgment, which can be time-consuming, subjective, and prone to inconsistencies, often leading to delayed diagnosis or false positives.

The Solution

The motivation behind this project was to design a data-driven and objective diagnostic support system that can assist healthcare professionals by improving diagnostic accuracy and reducing uncertainty. By leveraging deep learning models, the system aims to identify autism-related patterns at an early stage, making the diagnostic process faster and more reliable.

Technical Implementation

In this project, a Convolutional Neural Network (CNN) was developed using TensorFlow and Keras. The model was trained on 5,000+ datasets, which include relevant features associated with Autism Spectrum Disorder. CNNs are particularly effective in identifying complex and non-linear patterns in data, making them well-suited for medical prediction tasks.

Results & Impact

After training and evaluation, the model achieved a prediction accuracy of 94%, demonstrating strong diagnostic capability. Additionally, the system successfully reduced false positive rates by 15%, which is critical in avoiding unnecessary stress, misdiagnosis, and incorrect treatment plans.

By improving diagnostic precision, this project contributes to early autism detection, enabling healthcare professionals and caregivers to initiate timely intervention strategies. The system is intended to act as a decision-support tool, assisting clinicians rather than replacing professional medical judgment.

Overall, this project highlights the potential of AI-driven predictive analytics in healthcare, specifically in neurodevelopmental disorder diagnosis, enhancing accuracy, consistency, and efficiency in medical decision-making.

Face Recognition System

Attendance system using OpenCV recognizing 100+ faces in <2s with 95% accuracy.

OpenCV Python Pandas

The Face Recognition Attendance System using ESP32 is an automated attendance management solution designed to replace traditional manual and biometric attendance methods, which are often time-consuming, error-prone, and inconvenient.

Motivation

In educational institutions and organizations, taking attendance manually consumes valuable time and can lead to proxy attendance or incorrect record keeping. The motivation behind this project was to develop a contactless, fast, and accurate attendance system that leverages computer vision and embedded systems. Especially after the COVID-19 period, the need for touch-free and automated systems became more important for hygiene, efficiency, and scalability.

How It Works

The system uses an ESP32 camera module to capture facial images of individuals in real time. These images are processed using Python and OpenCV, where facial features are detected and compared against a pre-stored database using the face_recognition library. Once a face is successfully recognized, the system automatically marks the attendance of the individual.

Performance & Tech Stack

The system is capable of recognizing 100+ faces in under 2 seconds, making it suitable for real-time classroom or office environments. Attendance data is stored and managed efficiently using Pandas and NumPy, enabling structured record-keeping and easy data analysis.

Impact

During evaluation, the system achieved an overall recognition accuracy of 95%, ensuring reliable performance in real-world conditions. By automating the attendance process, the system saves approximately 15 minutes per session, significantly improving productivity for instructors and administrators.

This project demonstrates how embedded hardware (ESP32) combined with computer vision and machine learning techniques can solve everyday administrative problems, highlighting the practical application of AI in building smart, efficient, and scalable systems.

Drug Discovery Platform

ML platform analyzing 10,000+ compound datasets. Improved efficiency by 30%.

ML Streamlit

The Drug Discovery and Management System is a web-based healthcare platform developed to address two major challenges in the pharmaceutical domain: medicine supply chain management and efficient drug discovery and accessibility.

The Problem

The idea for this project originated during the COVID-19 pandemic (2020), when governments and healthcare authorities faced serious difficulties in tracking medicine manufacturing, distribution, and availability. Despite sufficient production, medicines often failed to reach the right places at the right time due to the absence of a centralized monitoring system. This resulted in shortages, black marketing, panic buying, and lack of transparency.

The Solution

To address this issue, we designed a centralized digital platform that allows medicines to be tracked end-to-end, starting from the manufacturer, moving through retailers, and finally reaching local medical shops where customers can easily purchase them. This enables better visibility, accountability, and control over the pharmaceutical supply chain.

Drug Discovery & Analysis

In addition to management, the project also focuses on the drug discovery and search process. Traditional drug analysis and discovery methods are time-consuming and computationally expensive. To overcome this, we applied machine learning techniques on a dataset of 10,000+ drug compounds to speed up analysis and improve prediction efficiency. As a result, the system reduces drug analysis time by 40% and improves prediction efficiency by 30%, significantly accelerating the discovery process.

Intelligent Drug Search

A key feature of the system is its intelligent drug search functionality. When a user searches for a medicine (for example, a branded drug like Dolo), the system not only checks its availability in nearby locations but also automatically suggests:

  • Alternative branded medicines with the same composition
  • Generic medicines that are safe, approved, and more affordable

This feature is especially useful during shortages, as it reduces dependency on a single brand and promotes awareness of cost-effective generic options.

Tech Stack

The entire system is implemented using Python, with data processing handled by Pandas and NumPy, machine learning models developed using TensorFlow, and a user-friendly web interface built using Streamlit.
